# MATEO

MATEO fue mi primer proyecto exploratorio para entender cómo trabajar con herramientas actuales de IA aplicadas al desarrollo de productos digitales.

<img width="1366" height="909" alt="image" src="https://github.com/user-attachments/assets/97e7e706-308c-477c-bfb0-5838946fdc04" />

El objetivo no fue construir un producto final, sino aprender cómo combinar herramientas como v0, Vercel, Neon y un flujo asistido con IA para crear una primera versión funcional de un SaaS simple para PyMEs.

<img width="1366" height="2188" alt="image" src="https://github.com/user-attachments/assets/96fb64be-011d-4cb6-aa8e-dd9c8307618d" />

Este proyecto me permitió validar algo importante: tareas que antes requerían más tiempo, coordinación y equipo, hoy pueden abordarse de forma mucho más liviana para prototipos o MVPs de baja a media complejidad.

## Qué incluye

- Dashboard inicial para visualizar métricas del negocio.
- Registro y gestión básica de movimientos.
- Conexión con base de datos en Neon.
- Interfaz pensada para una PyME.
- Flujo simple entre frontend, base de datos y despliegue.


## En qué me enfoqué

- Entender el flujo de trabajo con IA aplicada al desarrollo.
- Explorar cómo construir una interfaz funcional con v0.
- Conectar una aplicación web con una base de datos.
- Desplegar una primera versión en Vercel.
- Evaluar el potencial de estas herramientas para construir MVPs simples.

## Stack

- Next.js
- TypeScript
- Tailwind CSS
- Neon / SQL
- Vercel
- v0

## Estado del proyecto

Proyecto exploratorio.
Fue parte de una primera etapa de aprendizaje sobre desarrollo asistido con IA, producto digital y construcción rápida de MVPs.
