# MATEO

MATEO es un prototipo fintech pensado para explorar una experiencia simple, clara y profesional para PyMEs.

<img width="1366" height="909" alt="image" src="https://github.com/user-attachments/assets/97e7e706-308c-477c-bfb0-5838946fdc04" />

El proyecto fue creado con AI tools a partir de prompts y luego ajustado con ediciones sobre el código. Uno de los focos principales de este trabajo fue conectar la aplicación a una base de datos alojada en Neon y entender mejor ese flujo de integración.

<img width="1366" height="2188" alt="image" src="https://github.com/user-attachments/assets/96fb64be-011d-4cb6-aa8e-dd9c8307618d" />




## En qué me enfoqué

- prototipado rápido de una interfaz funcional
- iteración sobre contenido y estructura del producto
- conexión de la app con una base de datos en Neon y uso de SQL
- aprendizaje práctico de un flujo web con frontend + backend 

## Stack

- Next.js
- TypeScript
- Tailwind CSS
- Neon/SQL

## Estado del proyecto

Prototipo en evolución, usado principalmente como práctica para explorar construcción de producto, edición de código e integración con base de datos.
